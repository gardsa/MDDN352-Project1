Your Weather Your Way

Aims

To provide a weather forecast experience that is personal, local, and easy. In offering users the ability to customise their own interface, 4C will strip all the irrelevant numbers and graphs that don't make sense and display what YOU want to see.

Target Audience

Everyone. The service 4C will provide is already used across many apps and websites all over the world. The point of difference is that it should appeal to everyone as they have control over what they want to see.

Technical Construction

4C will be built as a web application, meaning users will not have to download an app, and it can be used on as many devices as possible. This means it will be built using HTML5, CSS3 and Javascript. It will harness relevant Javascript libraries for the modular user interface functionality, as well as the APIs of Metservice and BloomSky to retrieve the weather data.

Visual Direction

While users will be able to select from a number of different colour schemes, the default scheme has been created using two contrasting tones. The green and purple will form the main UI, while the light grey adds some contrast for primary elements. White has been selected for the majority of textual elements.

Two sans-serif fonts will be used. This choice was fairly evident, as serif fonts do not fit into the simple, minimal and informative aesthetic 4C aims to achieve. Noto Sans will serve as the main font-family, used for paragraphs and smaller text elements. Roboto is a very strong bold font, and will be used for larger text elements, such as the temperature or rainfall weather components.

Modular Interface Functionality

Modular design requires breaking down a design into small parts (modules), designed each independently, and then later combining them into a larger system. Modular UI and UX patterns will be followed to create 4C.This means creating a structured UI system, and understanding how its components can be used to create the user flow. This will allow users to customise the interface to fit their needs.

INTERACT: Drag and drop the weather component modules to experience how 4C will utilise the Packery and Draggabilly Javascript libraries to achieve a modular UI.

Users will also be able to resize the modules, and select their primary weather component.
